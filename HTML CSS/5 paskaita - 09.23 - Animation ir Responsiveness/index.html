Transform

  • translate(x,y) - Nurodo elemento pasistūmimą.
  • scale(x,y) - Nurodo elemento dydžio kitimą.
  • rotate(z) / rotateX(x) / rotateY(y) - Sukimasis aplink ašį.

Animation

  • animation-name - animacijos vardas
  • animation-duration - animacijos trukmė
  • animation-iteration-count - animacijos iteracijų kiekis
  • animation-delay - animacijos uždelsimas
  • animation-play-state - animacijos būsena (vykdoma ar ne)
  • animation-direction - animacijos kryptis
  • animation-timing-function - kaip skirstyti animacijai skirtą laiką
  • animation-fill-mode - ką daryti pasibaigus animavimui (retai naudojamas)
  • animation - apjungia visus stilius į vieną

@keyframes

Keyframe suteikiame norimos animuoti animacijos pavadinimą (@keyframes animationName). Ir žemiau nurodome animacijos žingsnius, kuriuose nurodome animuojamo elemento kitimą.
